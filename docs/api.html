API Reference

All endpoints prefixed with /api/v1. Base URL: http://localhost:4317/api/v1 (dev).

Auth: Protected routes require a JWT session cookie (set by login) or Authorization: Bearer <token> header.

Error format:

{ "error": true, "message": "Human-readable message", "details": {} }

Authentication (Public)

POST /auth/register

Create account. Sets auth cookie on success.

// Request
{ "email": "user@example.com", "password": "securePassword123", "name": "Optional" }
// 201
{ "user": { "id": "...", "email": "...", "name": "..." } }
// 400 — Validation error
// 409 — Email exists
Implemented

POST /auth/login

Authenticate. Sets auth cookie.

// Request
{ "email": "user@example.com", "password": "securePassword123" }
// 200
{ "user": { "id": "...", "email": "...", "name": "..." } }
// 401 — Invalid credentials
Implemented

POST /auth/logout

Clear auth cookie.

// 200
{ "ok": true }
Implemented

GET /auth/me

Get current user from session.

// 200
{ "user": { "id": "...", "email": "...", "name": "...", "avatar": null } }
// 401 — Not logged in
Protected

POST /auth/refresh

Refresh JWT token. Not yet implemented.

Wagers (Protected)

GET /wagers

List wagers. Supports filters via query string.

// Query: ?status=pending&limit=20&offset=0&sort=createdAt
// 200
{ "wagers": [ { ... } ], "total": 42 }
Protected

POST /wagers

Create a wager. You must include yourself in participants.

// Request
{
  "title": "Run 5km under 25min",
  "description": "Loser buys dinner",
  "stakeAmount": "20.00",
  "stakeCurrency": "USD",
  "participants": ["user-id-1", "user-id-2"],
  "deadline": "2025-06-30T23:59:59Z"
}
// 201
{ "wager": { ... } }
Protected

GET /wagers/:id

Get wager with participants, proofs, votes, comments.

// 200
{ "wager": { "id": "...", "title": "...", "status": "PENDING", "participants": [], "proofs": [], "votes": [], "comments": [] } }
// 404 — Not found
Protected

PATCH /wagers/:id

Update wager (creator only). Partial update.

// Request
{ "title": "Updated title", "status": "CANCELLED" }
// 200
{ "wager": { ... } }
// 403 — Not creator
Protected

DELETE /wagers/:id

Cancel wager (soft delete, sets status CANCELLED). Creator only.

// 204 — No content
// 403 — Not creator
Protected

POST /wagers/:id/participants

Add participant. Not yet implemented.

Planned

Proofs (Protected)

Votes (Protected)

Health

GET /health

Health check. Returns 503 if DB disconnected.

// 200
{ "status": "ok", "db": "connected", "uptime": 12345, "timestamp": "..." }
// 503 — DB down
Public

Status Codes

CodeMeaning
200OK
201Created
204No Content (DELETE)
400Validation error
401Unauthorized
403Forbidden (not owner/participant)
404Not found
409Conflict (duplicate vote, existing email)
429Rate limit exceeded
500Internal server error
501Not implemented (auth not configured)
503Service unavailable (DB down)
