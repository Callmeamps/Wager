Quick Start Guide

Get Wager running on your machine in under 5 minutes.

Prerequisites

  • Node.js 22+
  • npm
  • Git

1. Clone & Install

git clone https://github.com/your-org/wager.git
cd wager
bash scripts/install.sh

2. Configure Environment

Copy the example env file and edit as needed:

cp .env.example .env

For local development with SQLite (no extra services needed):

DATABASE_URL="file:./prisma/data/wagers.db"
JWT_SECRET="my-secret-key-change-me"

3. Seed Demo Data (Optional)

npm run seed
# Login: alice@example.com / password123

4. Start the Server

npm run dev

Open http://localhost:4317 in your browser.

5. Create an Account

Click Register in the nav bar, or go to /register directly. Use the seeded account above or create a new one.

6. Make Your First Wager

  1. Log in
  2. Click + New Wager (or go to /wagers)
  3. Fill in the title, add participants, set a deadline
  4. Publish — your wager is now live!

Quick Tips

  • Use npm run check to type-check before committing
  • Use npm run build for production builds
  • Use npm start to serve the production build
Note: The default JWT secret in .env.example is a placeholder. Generate a real one for production: openssl rand -base64 32
